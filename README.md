hacking. This is a living document — updated as the lab grows.

## Virtual Machines

### Ubuntu
- Primary lab machine
- Used for tool deployment, network analysis, and security tasks

---

## Tools

### Wireshark
Network protocol analyzer. Used for capturing and inspecting 
live traffic, analyzing packet structure, and identifying 
anomalies in network communication.

### Cisco Packet Tracer
Network simulation tool. Used for designing and troubleshooting 
network topologies, practicing routing and switching concepts 
aligned with CCNA and CyberOps curriculum.

### Metasploit Framework
Penetration testing framework. Used in controlled lab 
environments for understanding exploit development, vulnerability 
assessment, and attack simulation.
